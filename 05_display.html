Elementos inline, block, inline-block e floats

Utilizamos as propriedades display e float para controlar a forma na qual um elemento é posicionado na página e como os seus filhos são posicionados.

Elementos inline

Os elementos inline são dispostos um ao lado do outro (no nosso padrão, da esqueda para a direita). O seu tamanho não pode ser controlado pelas propriedades width e height.

Exemplos de elementos inline:

  • a
  • emph
  • strong
  • span

Com CSS, podemos marcar um elemento para ser inline com a propriedade display: inline.

Elementos block

Os elementos block são sempre dispostos sozinhos em uma linha, ocupando todo o espaço horizontal disponível. Ao adicionar múltiplos elementos block em uma página, estes acabam ficando empilhados.

Exemplos de elementos block:

  • p
  • h1, h2, ...
  • div, main, footer
  • ol, ul

Elementos inline-block

Os elementos inline-block permitem a configuração de sua altura e largura no CSS (diferindo dos inline) e não forçam a quebra da linha como ocorre nos elementos block.

Exemplo de elemento inline-block: img

Floats

O uso de float: left e float: right permitem com que um elemento saia do fluxo normal de leiaute e seja posicionado à esquerda ou à direita do seu container, possibilitando com que texto e demais elementos inline sejam dispostos ao seu redor.
